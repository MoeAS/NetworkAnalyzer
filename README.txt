To make sure the app is functioning follow these steps:

1. Go to settings
2. Go to app manager (or apps or any other close name depending on android api version)
3. Select the app "Network Analyzer"
4. Select Permissions 
5. Enable all of the permissions assigned
6. Restart the app


APK file is provided if needed to test on a device, simply move the apk to a device or download it on the device used and install then enable the permissions needed
